## 阶段一：服务器环境搭建（详细操作指南）

### 前置条件
- 已有Debian服务器，可通过SSH连接
- 已有域名 sqnl8.cn，已完成备案
- 域名DNS已解析到服务器IP

### 1.1 连接服务器

**Windows用户：**
1. 下载并安装 MobaXterm（推荐，免费版即可）
   - 下载地址：https://mobaxterm.mobatek.net/download.html
2. 打开MobaXterm → 点击 Session → SSH
3. 输入服务器IP地址、用户名（root）、密码
4. 连接成功后会看到命令行终端

**Mac用户：**
1. 打开"终端"应用
2. 输入：`ssh root@你的服务器IP`
3. 输入密码

### 1.2 安装基础软件

连接服务器后，依次执行以下命令（一行一行复制粘贴执行）：

```bash
# 第1步：更新系统软件包
apt update && apt upgrade -y

# 第2步：安装基础工具
apt install -y git curl wget vim htop ufw software-properties-common

# 第3步：安装Python 3和pip
apt install -y python3 python3-pip python3-venv python3-dev

# 第4步：安装MySQL数据库
apt install -y mariadb-server mariadb-client

# 第5步：安装Redis缓存
apt install -y redis-server

# 第6步：安装Nginx Web服务器
apt install -y nginx

# 第7步：安装Certbot（用于免费SSL证书）
apt install -y certbot python3-certbot-nginx

# 第8步：安装Chromium浏览器（数据采集需要）
apt install -y chromium chromium-driver

# 第9步：验证安装
python3 --version
mysql --version
redis-server --version
nginx -v
```

### 1.3 配置MySQL数据库

```bash
# 第1步：启动MySQL并设置开机自启
systemctl start mariadb
systemctl enable mariadb

# 第2步：安全配置MySQL
mysql_secure_installation
# 按提示操作：
# Enter current password for root: 直接按回车（初始没有密码）
# Switch to unix_socket authentication: N
# Set root password? Y → 输入一个强密码（记住这个密码！）
# Remove anonymous users? Y
# Disallow root login remotely? Y
# Remove test database? Y
# Reload privilege tables? Y

# 第3步：登录MySQL创建数据库和用户
mysql -u root -p
# 输入刚才设置的密码

# 在MySQL命令行中执行：
CREATE DATABASE juyunbang DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'juyunbang'@'localhost' IDENTIFIED BY '替换为你的密码';
GRANT ALL PRIVILEGES ON juyunbang.* TO 'juyunbang'@'localhost';
FLUSH PRIVILEGES;
EXIT;
```

**请记住以下信息（后面要用）：**
- 数据库名：juyunbang
- 数据库用户名：juyunbang
- 数据库密码：你设置的密码

### 1.4 配置Redis

```bash
# Redis安装后默认已启动，确认状态
systemctl start redis-server
systemctl enable redis-server

# 验证Redis运行正常
redis-cli ping
# 应该返回 PONG
```

### 1.5 配置域名DNS解析

在你的域名管理后台（域名注册商那里）添加DNS解析记录：

| 主机记录 | 记录类型 | 记录值 |
|---------|---------|--------|
| @ | A | 你的服务器IP |
| api | A | 你的服务器IP |

这样 `sqnl8.cn` 和 `api.sqnl8.cn` 都指向你的服务器。

### 1.6 配置SSL证书

等DNS解析生效后（通常几分钟到几小时）：

```bash
# 使用Let's Encrypt免费证书
certbot --nginx -d sqnl8.cn -d api.sqnl8.cn
# 按提示输入邮箱
# 同意服务条款：Y
# 是否分享邮箱：N
# 选择是否自动跳转HTTPS：2（自动跳转）

# 验证证书自动续期
certbot renew --dry-run
```

### 1.7 配置Nginx

```bash
# 创建Nginx配置文件
vim /etc/nginx/sites-available/juyunbang
```

写入以下内容：

```nginx
server {
    listen 80;
    server_name sqnl8.cn api.sqnl8.cn;
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name api.sqnl8.cn;

    ssl_certificate /etc/letsencrypt/live/sqnl8.cn/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/sqnl8.cn/privkey.pem;
    ssl_protocols TLSv1.2 TLSv1.3;

    # API代理到Flask后端
    location /api/ {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_connect_timeout 60s;
        proxy_read_timeout 120s;
    }

    # 健康检查
    location /health {
        proxy_pass http://127.0.0.1:5000/health;
    }

    location / {
        return 200 '{"service": "juyunbang-api", "status": "running"}';
        add_header Content-Type application/json;
    }
}

server {
    listen 443 ssl http2;
    server_name sqnl8.cn;

    ssl_certificate /etc/letsencrypt/live/sqnl8.cn/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/sqnl8.cn/privkey.pem;
    ssl_protocols TLSv1.2 TLSv1.3;

    location / {
        root /var/www/juyunbang;
        index index.html;
    }
}
```

```bash
# 启用配置
ln -sf /etc/nginx/sites-available/juyunbang /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default

# 测试配置
nginx -t

# 重启Nginx
systemctl restart nginx
systemctl enable nginx
```

### 1.8 配置防火墙

```bash

ufw allow 80     # HTTP
ufw allow 443    # HTTPS
ufw enable
# 确认输入 y
```

```bash

cd /opt/juyunbang/backend

```

### 1.10 配置七牛云

1. 注册七牛云账号：https://www.qiniu.com
2. 实名认证
3. 进入"对象存储" → "新建空间"
   - 空间名称：juyunbang
   - 存储区域：选离你服务器最近的
   - 访问控制：公开
4. 记录以下信息：
   - Access Key（在个人中心 → 密钥管理）
   - Secret Key
   - 存储空间名称：juyunbang
   - 绑定自定义域名（可选，用七牛云提供的测试域名也可以先用）

## UI/UX 设计方案

### 设计理念
- 品牌色：渐变紫蓝色系（#667eea → #764ba2），区别于竞品的红色系，更显高端
- 卡片化设计：所有数据以圆角卡片呈现，层次清晰
- 微动效：数字跳动、列表滑入、进度条填充
- 数据可视化：ECharts精美图表，折线图/柱状图/雷达图/环形图

### 配色系统

```
主色：#667eea（品牌紫蓝）
辅助色：#764ba2（深紫）
强调色：#f093fb（亮粉，用于热度高亮）
成功色：#2dce89（上涨/正向）
警告色：#fb6340（下降/异动）
背景色：#f4f5f7（浅灰白，柔和不刺眼）

深色模式：
```

### 首页视觉效果
- 顶部渐变紫蓝色头部，品牌名"剧云榜"居中
- 搜索栏：毛玻璃半透明效果
- 平台切换：胶囊形Tab，选中态带品牌色底

- 热度条形图：渐变紫蓝色进度条

- 热度走势用平滑折线图，带渐变填充区域
- 播放量用双轴图（柱状图+折线图叠加）

- 日历选择器：自定义日历组件，有数据的日期带小圆点标记
- 榜单Tab：滑动切换，选中态底部渐变下划线
- 每条数据：紧凑卡片式，包含迷你趋势火花线图（sparkline）

### 个人中心页
- 顶部圆形头像 + 昵称
- 追剧统计：三个圆形进度环（在追/想看/看完）
- 功能列表：图标+文字+箭头，分组展示
