# 第八册：小程序配置与发布

> **本册在哪操作**：
> - **本地电脑**（Windows/Mac，用微信开发者工具）
> - **浏览器**（打开微信公众平台后台）
>
> **目标**：配置微信公众平台的合法域名、在本地微信开发者工具里改 AppID 和 API 地址、真机预览、最终提交审核发布。

---

## 1. 微信公众平台配置（浏览器操作）

### 1.1 登录后台

浏览器打开 https://mp.weixin.qq.com → 用小程序管理员账号扫码登录。

### 1.2 配置服务器域名（最关键）

左侧菜单 → **开发管理** → **开发设置** → 向下滚动到 **"服务器域名"** 区域。

点击 **"修改"**（每月最多修改 5 次，填错会影响这个月的修改次数，**想清楚再填**）。

分别在以下三个字段里填入域名（**必须 https 开头，不能有结尾斜杠**）：

| 字段 | 必须填 | 示例 |
|---|---|---|
| **request 合法域名** | `https://<你的API域名>` | `https://api.nailao.asia` |
| **uploadFile 合法域名** | `https://<你的API域名>` | `https://api.nailao.asia` |
| **downloadFile 合法域名** | `https://<你的七牛云CDN域名>` （如果用了七牛云） | `https://cdn.example.com` |

**没有七牛云就不填 downloadFile**，留空即可（只要不用图片上传功能不影响核心）。

填完 → 点保存 → 会弹窗要求**管理员微信扫码确认**，扫码后生效。

### 1.3 验证域名配置生效

保存成功后，页面上这三个字段会显示你刚填的域名。**没显示就是没保存成功**，重新操作。

## 2. 安装微信开发者工具（本地电脑）

### 2.1 下载

打开 https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

选择对应的系统版本：
- **Windows 64**：exe 安装包
- **macOS**：dmg 安装包（注意区分 Intel / Apple Silicon M1/M2）

### 2.2 安装与登录

运行安装包 → 一路"下一步" → 安装后打开 → **用你的小程序管理员微信扫码登录**。

---

## 3. 把小程序代码放到本地

有两种方式：

### 方式 A（推荐）：本地 git clone

在 **本地电脑** 打开终端/PowerShell：

📍 **位置**：本地电脑

```
cd 想存代码的目录
git clone https://github.com/nailao11/juyunbang.git
cd juyunbang/miniapp
```

这样本地电脑的 `juyunbang/miniapp/` 就是完整的小程序代码。

### 方式 B：从服务器用 scp 拉下来

📍 **位置**：本地电脑

```
scp -r root@<你的服务器IP>:/opt/rejubang/miniapp ./miniapp
```

> ⚠️ **不要用 U 盘在服务器和本地间来回拷贝**——Linux 和 Windows 的换行符不同，可能导致 `.json` 文件解析失败。用 git 或 scp 最安全。

## 4. 修改小程序代码里的两处配置

### 4.1 修改 AppID

用任何代码编辑器（VS Code、记事本都行）打开 `miniapp/project.config.json`，找到：

```json
{
  ...
  "appid": "wxTODO_REPLACE_WITH_REAL_APPID",
  ...
}
```

把 `"appid"` 的值替换为你在第一册记下的真实 AppID：

```json
{
  ...
  "appid": "<你的小程序AppID>",
  ...
}
```

保存文件。

### 4.2 修改 API 地址

打开 `miniapp/app.js`，最开头能看到：

```javascript
App({
  globalData: {
    userInfo: null,
    token: null,
    baseUrl: 'https://api.nailao.asia/api/v1',
    ...
  },
  ...
})
```

把 `baseUrl` 替换为你的 API 域名（**路径后缀 `/api/v1` 不要改**）：

```javascript
baseUrl: 'https://<你的API域名>/api/v1',
```

比如：

```javascript
baseUrl: 'https://api.nailao.asia/api/v1',
```

保存文件。

## 5. 在微信开发者工具中导入项目

### 5.1 导入

打开微信开发者工具 → 点击 **"导入项目"** →
- **项目目录**：选择本地电脑上的 `juyunbang/miniapp` 目录（**是 miniapp 目录本身，不是 juyunbang**）
- **AppID**：工具会自动读取 `project.config.json` 里的 AppID，**不要在这里手动改**
- **项目名称**：随意，如 "热剧榜"
- 点 **"确定"**

### 5.2 编译与运行

导入后工具会自动编译项目。成功的话：
- 左侧模拟器能看到小程序首页（有顶部渐变 + 实时热度榜）
- 底部控制台没有红色报错

### 5.3 常见启动报错排查

**"不在以下 request 合法域名列表中"**
- 回到 1.2 确认 **request 合法域名** 已添加
- 本地 devtools 工具栏右上角 **"详情"** → **"本地设置"** → 勾选 **"不校验合法域名..."**（仅开发时用，不影响提审）

**"request:fail url not in domain list"**
- 同上

**"接口请求超时 / 失败"**
- 后端 API 没跑起来。在服务器上跑 `systemctl status rejubang-api` 确认
- 本地用浏览器访问 `https://<你的API域名>/health` 应该返回 `{"status":"ok"}`

## 6. 真机预览测试

### 6.1 生成预览二维码

工具栏 → **"预览"** 按钮 → 会弹出二维码。

### 6.2 用手机微信扫码

- **管理员本人的微信**扫码可以直接进
- 其他人扫码需要先把他们添加为 **"体验成员"**（小程序管理后台 → 成员管理 → 体验成员）

### 6.3 核心功能自测

建议测试以下路径，确保 API 可达：

| 页面 | 测试动作 | 期望结果 |
|---|---|---|
| 首页 | 下拉刷新 | 榜单能刷新 |
| 首页 | 切换平台 tab | 不同平台显示不同榜单 |
| 搜索 | 输入任意剧名 | 能看到搜索建议或结果 |
| 数据中心 | 切换日榜/周榜/月榜 | 不报错（可能是空榜） |
| 剧集详情 | 点击首页任一剧 | 能显示详情页 |
| 我的 | 查看设置 / 主题切换 | 主题能切换 |

## 7. 上传代码并提交审核

### 7.1 上传

微信开发者工具 → **"上传"** 按钮 → 填入：
- **版本号**：例如 `1.0.0`
- **项目备注**：例如 `首次发布`

点 **"上传"** → 等待上传完成。

### 7.2 在微信公众平台提审

浏览器打开 https://mp.weixin.qq.com → **版本管理** → **开发版本** → 找到刚上传的版本 → 点 **"提交审核"**。

按提示填写：
- **类目**：选合适的（工具 / 效率 / 图书影音 等）
- **功能描述**：简短说明
- **测试账号**：如无登录要求可选"不需要测试账号"

提交后等待审核（1~7 个工作日）。

### 7.3 审核通过后发布

审核通过 → 邮件和公众平台会通知 → 进 **版本管理** → **审核版本** → 点 **"发布"** → 正式上线。
