## 部署前你需要知道的事

本项目包含两部分：
- **后端**：部署在 **Linux 云服务器** 上（Python + Flask + MariaDB + Redis + Playwright）。
- **前端**：微信小程序，部署在 **你的电脑** 上的"微信开发者工具"里编译并提交审核。

两部分通过 HTTPS 接口通信，所以你需要：
1. **一台云服务器**（4核4G，Debian 12）
2. **一个域名**（做 HTTPS + 微信小程序合法域名要求）
3. **一台本地电脑**（Windows/Mac 均可，用来跑微信开发者工具）

## 一些必须先理解的约定

阅读文档时请记住以下几个约定，之后不再重复说明：

### 1. 命令提示符含义

每条命令前面会有一个"提示符"，告诉你这条命令应该在哪里执行：

| 提示符 | 含义 |
|---|---|
| `[本地] $` | 在**你自己电脑**的终端里执行（Windows PowerShell / Mac 终端） |
| `root@server:~#` | 在**云服务器**上执行，当前目录是 `/root` |
| `root@server:/opt/rejubang/backend#` | 在**云服务器**上执行，当前目录是 `/opt/rejubang/backend` |

**看到命令时，先看提示符里的目录，不对就先 `cd` 过去再执行。**

### 2. 需要你替换的占位符

文档里所有尖括号包裹的内容 `<xxx>` 都是占位符，需要替换成你自己的值：

| 占位符 | 含义 | 举例 |
|---|---|---|
| `<你的服务器IP>` | 云服务器公网 IP | `123.45.67.89` |
| `<你的域名>` | 你买的主域名 | `nailao.asia` |
| `<你的API域名>` | 通常是 `api.<你的域名>` | `api.nailao.asia` |
| `<你的小程序AppID>` | 微信小程序 AppID | `wx1234567890abcdef` |

**替换时不要保留尖括号。** 比如 `<你的域名>` 替换后应该是 `nailao.asia`，不是 `<nailao.asia>`。

### 3. 已经预生成好的密钥

以下密钥与密码文档里已经帮你**生成好了固定值**，直接复制粘贴即可，不需要你再重新生成、也不需要你记住：

- `SECRET_KEY`（Flask 会话密钥）
- `JWT_SECRET_KEY`（JWT 签名密钥）
- `DB_PASSWORD`（项目数据库用户密码）

这些值只在服务器本地使用，不对外暴露，用预设值不影响安全性。

### 4. 随时可以中断重来

部署流程里任何一步出错都可以定位到 [10-常见问题FAQ.md](./10-常见问题FAQ.md) 或重新执行对应步骤。已完成的步骤（如装系统包、建库）重复执行大多数是幂等的（不会造成破坏）。
