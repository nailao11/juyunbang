## 📝 阅读约定（通用）

本指南每册都会遵循以下约定：

- **命令前的提示符**就是告诉你"在哪里执行"：
  - `[本地] $` = 本地电脑
  - `root@server:~#` = 服务器 root 用户在 `/root`
  - `root@server:/opt/rejubang/backend#` = 服务器 root 用户在 `/opt/rejubang/backend`
- **尖括号 `<xxx>`** 是占位符，要替换成你的真实值（替换后不要保留尖括号）
- **密码里的 `!` 和 `$`** 如果需要在 shell 里用，始终用**单引号**包起来（`'RootDB_Pass_2026_NaiLao!'`），避免特殊字符被 shell 解析
