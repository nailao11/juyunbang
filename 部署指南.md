# 剧云榜 完整部署指南

> 4核4GB服务器 + 40GB云硬盘 部署方案
>
> 采集架构：Playwright 无头浏览器渲染页面，提取四大平台真实站内热度值
> 采集策略：只采集近30天内上线的新剧，每15分钟增量采集，同剧同平台10分钟内不重复

---

## 第一步：系统基础环境

> 每次 SSH 登录后，先确认当前路径再操作

```bash
# 确认当前位置（安卓SSH重连后可能回到 /root）
pwd

# 安装系统基础包
apt update && apt install -y python3 python3-venv python3-pip git nginx mariadb-server redis-server curl
```

---

## 第二步：配置 Swap（防止 Chromium 内存不足被杀）

```bash
# 检查是否已有Swap
swapon --show

# 如果没有输出，创建2GB Swap
fallocate -l 2G /swapfile
chmod 600 /swapfile
mkswap /swapfile
swapon /swapfile

# 设为开机自动挂载
echo '/swapfile none swap sw 0 0' >> /etc/fstab

# 验证（应显示约2G）
free -h
```

> 为什么需要 Swap：Playwright 启动 Chromium 浏览器单实例占用 300-500MB。
> 4GB 内存同时跑 Flask + MariaDB + Redis + Chromium，峰值可能超 4GB。
> 加 Swap 后系统有缓冲，不会因为内存不足直接杀进程。

## 第五步：安装 Playwright 浏览器

```bash
# 确认在后端目录

# 安装 Chromium 浏览器（约150MB）

# 安装 Chromium 运行所需的系统库
```

## 第七步：配置数据库

```bash
# 启动 MariaDB
systemctl start mariadb
systemctl enable mariadb

# 创建数据库和用户（按提示输入root密码，首次安装密码为空直接回车）
mysql -u root -p <<'SQL'
CREATE DATABASE IF NOT EXISTS juyunbang DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER IF NOT EXISTS 'juyunbang'@'localhost' IDENTIFIED BY '替换为你的密码';
GRANT ALL PRIVILEGES ON juyunbang.* TO 'juyunbang'@'localhost';
FLUSH PRIVILEGES;
SQL

# 导入表结构
mysql -u juyunbang -p'替换为你的密码' juyunbang < /opt/juyunbang/backend/migrations/schema.sql

# 导入初始数据（平台信息 + 示例剧集）
mysql -u juyunbang -p'替换为你的密码' juyunbang < /opt/juyunbang/backend/migrations/seed_data.sql
```

## 第十步：配置 Nginx

```bash
# 复制配置文件
cp /opt/juyunbang/deploy/nginx.conf /etc/nginx/sites-available/juyunbang
ln -sf /etc/nginx/sites-available/juyunbang /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default

# 测试配置是否正确
nginx -t

# 如果报SSL证书不存在的错误，先注释掉SSL部分或申请证书（见下方说明）
# 启动 Nginx
systemctl restart nginx
systemctl enable nginx
```

> SSL 证书申请（如果还没有）：
> ```bash
> apt install -y certbot python3-certbot-nginx
> certbot --nginx -d api.sqnl8.cn -d sqnl8.cn
> ```

## 微信小程序配置

### 在微信开发者工具中：

1. **导入项目**：打开微信开发者工具 → 导入项目 → 选择 `/opt/juyunbang/miniapp` 目录（或本地开发机上的 miniapp 目录）

2. **修改 AppID**：编辑 `miniapp/project.config.json`，将 `wxTODO_REPLACE_WITH_REAL_APPID` 替换为你的真实小程序 AppID

3. **配置服务器域名**：
   - 登录 [微信公众平台](https://mp.weixin.qq.com) → 开发管理 → 开发设置 → 服务器域名
   - request 合法域名添加：`https://api.sqnl8.cn`

4. **修改 API 地址**：编辑 `miniapp/utils/request.js`，确认 `baseUrl` 为 `https://api.sqnl8.cn/api/v1`

5. **预览/上传**：
   - 开发者工具中点击"预览"可手机扫码测试
   - 确认无误后点击"上传"提交审核

## 常见问题

**Q: Playwright 安装报错？**
```bash
cd /opt/juyunbang/backend
./venv/bin/pip uninstall -y playwright
./venv/bin/pip install playwright==1.40.0
./venv/bin/playwright install chromium --force
./venv/bin/playwright install-deps chromium
```

**Q: 采集服务启动后没有数据？**
- 查看日志：`tail -50 /opt/juyunbang/logs/scheduler_$(date +%Y-%m-%d).log`
- 确认 dramas 表中有近30天 air_date 的剧集数据
- 单独测试：`cd /opt/juyunbang/backend && ./venv/bin/python test_crawl.py --platform tencent`

**Q: 内存不足进程被杀？**
```bash
# 确认Swap已启用
free -h
# 如果Swap行全是0，参照第二步创建Swap
```

**Q: API 502 Bad Gateway？**
```bash
# 检查API服务是否在运行
systemctl status juyunbang-api
# 查看错误日志
journalctl -u juyunbang-api -n 50
# 重启
systemctl restart juyunbang-api
```

**Q: SSH 断开重连后命令找不到？**
```bash
# 安卓SSH客户端重连后会回到 /root，需要重新切目录
cd /opt/juyunbang/backend
# 如果需要用 venv 里的命令
source venv/bin/activate
```
