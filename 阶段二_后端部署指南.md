## 第二步：安装Python依赖

```bash
# 进入后端目录
cd /opt/juyunbang/code/backend

# 创建Python虚拟环境
python3 -m venv venv

# 激活虚拟环境（激活后命令行前面会出现 (venv) 字样）
source venv/bin/activate

# 升级pip（避免安装报错）

# 安装所有依赖（使用国内镜像加速，需要几分钟）
```

**等待安装完成后，看到 `Successfully installed ...` 就是成功了。**

如果安装过程中出现红色报错提到 `lxml` 或 `cryptography`，执行以下命令后重试：
```bash
apt install -y libxml2-dev libxslt1-dev libffi-dev build-essential
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 第四步：配置环境变量

`.env` 文件因为安全原因不会随代码一起下载（被 `.gitignore` 排除），需要手动创建。

### 4.1 先配置七牛云域名（如果还没做的话）

在创建 `.env` 之前，你需要先确定七牛云的 CDN 域名。有两种方式：

**方式A：使用七牛云测试域名（推荐先用这个，快速上线）**
1. 登录七牛云 → 进入"对象存储" → 选择 `juyunbang` 空间
2. 点击"域名管理"，你会看到一个测试域名（类似 `xxx.test.clouddn.com`）
3. 复制这个域名，填入下面 `.env` 的 `QINIU_DOMAIN` 中
4. 注意：测试域名有限制（每日流量有限、30天后可能回收），正式上线前需换成自定义域名

**方式B：绑定自定义域名（正式上线时使用）**
1. 登录七牛云 → "对象存储" → 选择 `juyunbang` 空间 → "域名管理" → "绑定域名"
2. 输入你要绑定的域名，例如 `cdn.sqnl8.cn`
3. 七牛云会给你一个 CNAME 值（类似 `xxx.qiniudns.com`）
4. 去你的域名管理后台（域名注册商那里），添加 DNS 解析：

| 主机记录 | 记录类型 | 记录值 |
|---------|---------|--------|
| cdn | CNAME | 七牛云给你的CNAME值 |

5. 等待 DNS 生效后（通常几分钟），在七牛云后台点击"验证"
6. 验证成功后，`cdn.sqnl8.cn` 就可以用了

### 4.2 创建 `.env` 文件

复制粘贴以下整段命令到服务器执行（SECRET_KEY 和 JWT_SECRET_KEY 会自动生成随机值，无需手动填写）：

```bash
# 自动生成两个随机密钥
SK=$(python3 -c "import secrets; print(secrets.token_hex(32))")
JK=$(python3 -c "import secrets; print(secrets.token_hex(32))")

cat > /opt/juyunbang/code/backend/.env << EOF
# 服务器配置
FLASK_ENV=production
SECRET_KEY=$SK

# MySQL数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_NAME=juyunbang
DB_USER=juyunbang
DB_PASSWORD=替换为你的数据库密码

# Redis配置
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_DB=0

# 微信小程序配置
WX_APPID=替换为你的小程序AppID
WX_SECRET=替换为你的小程序AppSecret

# 七牛云配置
QINIU_ACCESS_KEY=替换为你的七牛云AccessKey
QINIU_SECRET_KEY=替换为你的七牛云SecretKey
QINIU_BUCKET=juyunbang
QINIU_DOMAIN=替换为你的七牛云域名

# JWT配置
JWT_SECRET_KEY=$JK
JWT_ACCESS_TOKEN_EXPIRES=86400
EOF
```

### 4.3 修改 `.env` 中的占位内容为你的实际信息

```bash
vim /opt/juyunbang/code/backend/.env
```

**你需要替换以下每一项为你的真实值：**

| 配置项 | 说明 | 如何获取 |
|--------|------|---------|
| SECRET_KEY | 服务器密钥 | **已自动生成，无需修改** |
| DB_PASSWORD | 数据库密码 | 阶段一步骤1.3中你为juyunbang用户设置的密码 |
| WX_APPID | 微信小程序AppID | 微信公众平台 → 开发管理 → 开发设置 |
| WX_SECRET | 微信小程序AppSecret | 微信公众平台 → 开发管理 → 开发设置 |
| QINIU_ACCESS_KEY | 七牛云AK | 七牛云 → 个人中心 → 密钥管理 |
| QINIU_SECRET_KEY | 七牛云SK | 七牛云 → 个人中心 → 密钥管理 |
| QINIU_DOMAIN | 七牛云CDN域名 | 上面步骤4.1中获取的域名（不带http://） |
| JWT_SECRET_KEY | JWT令牌密钥 | **已自动生成，无需修改** |

> **vim编辑器使用方法（最简单版）：**
> 1. 打开文件后，按 `i` 键进入编辑模式（左下角显示 `-- INSERT --`）
> 2. 用方向键移到要改的位置，删除旧内容，输入新内容
> 3. 改完后按 `Esc` 键退出编辑模式
> 4. 输入 `:wq` 然后按回车 → 保存并退出
> 5. 如果改错了想放弃，按 `Esc` 后输入 `:q!` 然后按回车 → 不保存退出

## 第七步：配置Gunicorn + Systemd（让后端在后台持续运行）

```bash
# 创建systemd服务文件
cat > /etc/systemd/system/juyunbang.service << 'EOF'
[Unit]
Description=JuYunBang Backend API
After=network.target mariadb.service redis-server.service

[Service]
Type=notify
User=root
Group=root
WorkingDirectory=/opt/juyunbang/code/backend
Environment="PATH=/opt/juyunbang/code/backend/venv/bin:/usr/bin"
ExecStart=/opt/juyunbang/code/backend/venv/bin/gunicorn -c gunicorn_config.py run:app
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
EOF
```

```bash
# 重新加载systemd配置
systemctl daemon-reload

# 启动后端服务
systemctl start juyunbang

# 设置开机自动启动
systemctl enable juyunbang

# 查看服务状态（应该显示 active (running)）
systemctl status juyunbang
```

**你应该看到绿色的 `active (running)` 字样。** 如果显示红色的 `failed`，执行以下命令查看详细错误：
```bash
journalctl -u juyunbang -n 50 --no-pager
```

## 常见问题

### Q: `mysql -u juyunbang -p` 报错 "Access denied"
A: 密码输错了。如果忘记了密码，用root重新设置：
```bash
mysql -u root -p
# 输入root密码后执行：
ALTER USER 'juyunbang'@'localhost' IDENTIFIED BY '你的新密码';
FLUSH PRIVILEGES;
EXIT;
```
然后更新 `/opt/juyunbang/code/backend/.env` 中的 `DB_PASSWORD`。

### Q: `pip install` 特别慢
A: 使用国内镜像加速：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: `systemctl status juyunbang` 显示 failed
A: 执行 `journalctl -u juyunbang -n 50 --no-pager` 查看详细错误，把输出截图给我。

### Q: `curl https://api.sqnl8.cn/...` 报错
A: 检查Nginx配置：
```bash
nginx -t
systemctl restart nginx
```
如果还是不行，把 `nginx -t` 的输出截图给我。

### Q: 七牛云测试域名过期了怎么办？
A: 按照上面步骤4.1的方式B绑定自定义域名，然后更新 `.env` 中的 `QINIU_DOMAIN`，再重启服务：
```bash
systemctl restart juyunbang
```
